# App Icon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the default Android template icon with a custom Grayout icon — concentric gray circles on a near-black background.

**Architecture:** Four new XML files (two vector drawables + two adaptive-icon wrappers), replacing 14 legacy files. No code changes, no manifest changes.

**Tech Stack:** Android VectorDrawable XML, Adaptive Icon XML (API 26+)

### Task 1: Create foreground vector drawable

**Files:**
- Create: `app/src/main/res/drawable/ic_grayout_foreground.xml`

- [ ] **Step 1: Create the foreground drawable**

Create `app/src/main/res/drawable/ic_grayout_foreground.xml` with four concentric circles centered at (54, 54) on a 108dp viewport. Circles are drawn as two-arc paths since VectorDrawable has no `<circle>` element. The path formula for a circle at (cx, cy) with radius r is: `M cx-r,cy a r,r 0 1,1 2r,0 a r,r 0 1,1 -2r,0 z`

```xml
<?xml version="1.0" encoding="utf-8"?>
<vector xmlns:android="http://schemas.android.com/apk/res/android"
    android:width="108dp"
    android:height="108dp"
    android:viewportWidth="108"
    android:viewportHeight="108">

    <!-- Outer ring: r=28, stroke, alpha 0.2 -->
    <path
        android:pathData="M26,54a28,28 0 1,1 56,0a28,28 0 1,1 -56,0z"
        android:fillColor="#00000000"
        android:strokeColor="#D4D4D4"
        android:strokeWidth="2"
        android:alpha="0.2" />

    <!-- Mid ring: r=20, stroke, alpha 0.5 -->
    <path
        android:pathData="M34,54a20,20 0 1,1 40,0a20,20 0 1,1 -40,0z"
        android:fillColor="#00000000"
        android:strokeColor="#D4D4D4"
        android:strokeWidth="2"
        android:alpha="0.5" />

    <!-- Inner ring: r=12, stroke, alpha 0.8 -->
    <path
        android:pathData="M42,54a12,12 0 1,1 24,0a12,12 0 1,1 -24,0z"
        android:fillColor="#00000000"
        android:strokeColor="#D4D4D4"
        android:strokeWidth="2"
        android:alpha="0.8" />

    <!-- Center dot: r=4, solid fill -->
    <path
        android:pathData="M50,54a4,4 0 1,1 8,0a4,4 0 1,1 -8,0z"
        android:fillColor="#D4D4D4" />

</vector>
```

- [ ] **Step 2: Commit**

```
git add app/src/main/res/drawable/ic_grayout_foreground.xml
git commit -m "feat(icon): add foreground vector drawable with concentric circles"
```

### Task 3: Create adaptive icon wrappers in mipmap-anydpi-v26

**Files:**
- Create: `app/src/main/res/mipmap-anydpi-v26/ic_launcher.xml`
- Create: `app/src/main/res/mipmap-anydpi-v26/ic_launcher_round.xml`

- [ ] **Step 1: Create the mipmap-anydpi-v26 directory**

```bash
mkdir -p app/src/main/res/mipmap-anydpi-v26
```

- [ ] **Step 2: Create ic_launcher.xml**

Create `app/src/main/res/mipmap-anydpi-v26/ic_launcher.xml`:

```xml
<?xml version="1.0" encoding="utf-8"?>
<adaptive-icon xmlns:android="http://schemas.android.com/apk/res/android">
    <background android:drawable="@drawable/ic_grayout_background" />
    <foreground android:drawable="@drawable/ic_grayout_foreground" />
    <monochrome android:drawable="@drawable/ic_grayout_foreground" />
</adaptive-icon>
```

- [ ] **Step 3: Create ic_launcher_round.xml**

Create `app/src/main/res/mipmap-anydpi-v26/ic_launcher_round.xml` — identical content to `ic_launcher.xml`:

```xml
<?xml version="1.0" encoding="utf-8"?>
<adaptive-icon xmlns:android="http://schemas.android.com/apk/res/android">
    <background android:drawable="@drawable/ic_grayout_background" />
    <foreground android:drawable="@drawable/ic_grayout_foreground" />
    <monochrome android:drawable="@drawable/ic_grayout_foreground" />
</adaptive-icon>
```

- [ ] **Step 4: Commit**

